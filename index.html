02 — Quick Start

02 — Inicio Rápido

Quick Start

Inicio Rápido

📦 InstallationInstalación
  1. 1 Copy the desired RSK_PluginName.js files into your project's /js/plugins/ folder. Copia los archivos RSK_NombrePlugin.js en la carpeta /js/plugins/ de tu proyecto.
  2. 2 Open RPG Maker MZ → Plugin Manager → Add Plugin → select the RSK plugin. Abre RPG Maker MZ → Administrador de Plugins → Añadir Plugin → selecciona el plugin RSK.
  3. 3 Configure parameters in the Plugin Manager panel and press OK. Configura los parámetros en el Administrador de Plugins y pulsa OK.
  4. 4 Save your project. The plugin activates immediately in Test Play. Guarda el proyecto. El plugin se activa inmediatamente en Prueba de Juego.
⚙️ Load OrderOrden de Carga

Place RSK plugins AFTER VisuStella Core Engine (if used). Order between other RSK plugins is flexible unless noted in individual docs.

Coloca los plugins RSK DESPUÉS del VisuStella Core Engine (si lo usas). El orden entre otros plugins RSK es flexible salvo que se indique lo contrario.

RSK_SuperTool → RSK_AtmosphereStudio → RSK_NatureStudio
RSK_MovementStudio → RSK_JumpStudio → RSK_SeamlessWorld
